# Optimizing Your Listing

## Discovery

Skill Shope users find tools by browsing categories, searching keywords, and filtering by type. Optimize for all three.

## Search Optimization

| Factor | What to do |
|--------|-----------|
| **Name** | Include the primary keyword: "Postgres MCP Server" not "DBHelper" |
| **Description** | Front-load the value prop. First 100 characters show in search results |
| **Tags** | Add 3-8 relevant, specific tags. Include the technology, use case, and tool type |
| **Category** | Pick the single most relevant category — this drives browse traffic |
| **Compatibility** | List every tool you support — users filter by this |

## Building Trust

| Signal | How to earn it |
|--------|---------------|
| **Verified Publisher badge** | Contact Skill Shope admins after publishing quality listings |
| **Verified Skill badge** | Admins verify skills that meet quality and security standards |
| **Featured badge** | High-quality, popular skills get featured on the homepage |
| **Reviews** | Encourage users to leave reviews — social proof drives purchases |
| **Download count** | Grows organically as users install — no shortcuts |

## Pricing Strategy

| Strategy | When to use |
|----------|-----------|
| **Start free** | New publisher, first skill, building reputation. Get reviews and downloads first |
| **Free + paid tier** | Offer a free basic version and a paid premium version with more features |
| **Paid from day one** | Established publisher with proven track record and unique tool |
| **Price anchoring** | $4.99-$9.99 is the sweet spot for individual developers. $19.99+ for team/enterprise tools |

## Traps to Avoid

- **Pricing too high with no reviews** — Buyers won't take a $19.99 risk on an unreviewed skill
- **Empty description** — Listings with only a name and one-liner get skipped
- **No install command** — Friction kills conversion. Make installation copy-paste simple
- **Listing the same skill multiple times** — Duplicate listings violate content policy and get removed
