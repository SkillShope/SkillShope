# RoughInHub

**The registry for the agentic era.**

RoughInHub is a marketplace where developers discover, review, and install AI blueprints, MCP servers, and agent configurations for Claude Code, Codex, Cursor, and other AI coding assistants. Publishers list their tools and earn from their work.

[roughinhub.com](https://roughinhub.com)

---

## Features

- **Browse & Search** — Find AI blueprints by category, type, compatibility, and rating
- **Verified Publishers** — Trust badges for vetted publishers and reviewed blueprints
- **One-Command Install** — Every listing includes a copy-paste install command
- **Secure Payments** — Stripe Checkout with no raw card data handling
- **Publisher Payouts** — Automatic 85% revenue share via Stripe Connect
- **Reviews & Ratings** — Community-driven trust through genuine reviews

## For Publishers

List your AI blueprints, MCP servers, and agent configs on RoughInHub. Set your price (or publish for free), connect your Stripe account, and start earning.

See the [Publisher Guide](https://roughinhub.com/about) for details.
